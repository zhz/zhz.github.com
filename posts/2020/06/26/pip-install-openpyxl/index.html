python 安装 openpyxl

2020-06-26
  • 编程

pandas 写 excel 需要 openpyxl 的支持，直接 pip3 install openpyxl 提示未找到相应版本，只好到 https://pypi.python.org/pypi 去主动下载包，pip 支持直接安装 .tar.gz 和 .whl 的包，不过安装 openpyxl 过程中需要提前安装 jdcal 和 et_xmlfile 两个包，直接搜索下来三个包，依次 pip3 install 即可。
